Pinout: El sistema está pensado para utilizar directamente las salidas integradas del raspi, de esa forma, no hay que utilizar hardware específico.

Se utilizan los mismos pines para el ESP y para el ATiny85 
ESP: 
Reset: GPIO18 
Gpio0ESP(se usa para cambiar el modo de ejecución): GPIO17 ,SCK en Attiny

| TX (GPIO14) | RX | SDO |
| --- | --- | --- |
| Pin Raspy | Pin esp | PinAttiny |

| RX (GPIO15) | TX | Sdi |
| --- | --- | --- |

En el caso del ATTiny85, no se hacen scripts en Py para entrar y salir del modo prog, se edita el archivo /usr/local/etc/avrdude.conf

Editando esta parte

programmer # raspberry_pi_gpio
id                     = "raspberry_pi_gpio";
desc                   = "Raspberry Pi GPIO via sysfs/libgpiod";
type                   = "linuxgpio";
prog_modes             = PM_ISP;
connection_type        = linuxgpio;
reset                  = 18;
sck                    = 17;
sdo                    = 14;
sdi                    = 15;
;
